## 実装された機能

  1. ダッシュボード (`/dashboard`)

### ヘッダー (`components/dashboard/VenueAdmin/Header.tsx`)
- **現在操作中の挙式名表示**:
  - Zustandストアから選択中の挙式を取得
  - ヘッダー左側に青色のバッジとして表示
  - 挙式が選択されていない場合は「挙式を選択してください」と表示

- **Command Palette (Cmd+K)**:
  - 引き続き利用可能

### ダッシュボード (`app/dashboard/page.tsx`)
- **カウントダウン機能**:
  - 次の挙式までの日数をリアルタイムで表示
  - グラデーション背景のカードで視覚的に強調
  - 挙式名、日時も表示

- **統計カード**:
  - 総ゲスト数（青色）
  - 未回答ゲスト数（黄色）
  - アレルギーあり人数（赤色）
  - 確定ゲスト数（緑色）
  - Framer Motionによるアニメーション

- **クイックアクション**:
  - 挙式設定、ゲスト管理、席次表・配席へのクイックリンク

### 挙式基本設定 (`app/dashboard/settings/page.tsx`)
- **フォーム実装**:
  - `react-hook-form` による状態管理
  - `zod` によるバリデーション
  - リアルタイムエラー表示

- **入力フィールド**:
  1. 新郎氏名（必須）
  2. 新婦氏名（必須）
  3. 挙式日（必須、date input）
  4. 挙式時間（必須、time input）
  5. 会場名（必須）
  6. ウェルカムメッセージ（任意、textarea）

- **保存機能**:
  - 保存ボタンクリック時にToast通知を表示
  - Zustandストアを更新
  - 未変更時は保存ボタンを無効化

- **UI特徴**:
  - セクションごとに区切り線で整理
  - アイコン付きのセクションタイトル
  - 余白を多めに取り、圧迫感のないレイアウト
  - エラー時の視覚的フィードバック（赤色背景）

## デザインシステム

### カラーパレット
- **プライマリカラー**: 青色（`blue-600`, `blue-50`, `blue-100`）
- **背景色**: 白（`white`）、薄いグレー（`gray-50`）
- **テキスト**: 濃いグレー（`gray-900`）、中程度のグレー（`gray-600`）
- **境界線**: 薄いグレー（`gray-200`）

### フォント
- **主要フォント**: Noto Sans JP（可読性重視のサンセリフ体）
- **フォントサイズ**: タイトル `text-2xl`、本文 `text-sm`、説明文 `text-xs`

### 余白（Padding/Margin）
- **ページコンテンツ**: `p-8`（2rem）
- **サイドバーメニュー**: `p-6`（1.5rem）
- **カード内**: `p-6` または `p-8`
- **フォーム要素間**: `space-y-6` または `space-y-8`

### アニメーション
- **Framer Motion**: フェードイン、スケール、Y方向の移動
- **トランジション**: `transition-all duration-200`

## 技術スタック

- **Framework**: Next.js 14 (App Router)
- **Form Management**: react-hook-form
- **Validation**: zod
- **Toast Notification**: sonner
- **State Management**: Zustand
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Tailwind CSS
